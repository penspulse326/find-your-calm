# Find Your Calm - 互動式心靈評測遊戲

這是一個基於 Vue 3 開發的「文字冒險遊戲（AVG）」風格焦慮評測工具。透過情境代入感幫助用戶評估焦慮程度，並提供科學建議。

## 🌟 專案特點

- **沉浸式體驗**：採用視覺小說風格，搭配立繪與打字機效果。
- **科學依據**：指標參考 GAD-7 焦慮量表。
- **反應式設計**：根據使用者選擇動態切換角色情緒與場景氛圍。

## 🛠 技術棧

- **框架**：Vue 3 (Composition API)
- **構建工具**：Vite + TypeScript
- **樣式**：Tailwind CSS
- **動畫**：@vueuse/motion + CSS Animation
- **狀態管理**：Pinia

## 🚀 快速開始

```bash
# 安裝依賴
pnpm install

# 啟動開發伺服器
pnpm dev

# 打包專案
pnpm build
```
