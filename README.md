# Find Your Calm - 互動式心靈評測遊戲

這是一個基於 Vue 3 開發的「文字冒險遊戲（AVG）」風格焦慮評測工具。透過情境代入感幫助用戶評估焦慮程度，並提供科學建議。

## 🌟 專案特點

- **沉浸式體驗**：採用視覺小說風格，搭配靈活的角色立繪與情境變化。
- **對白動態引擎**：自訂腳本系統，支援對白延遲、逼真的打字機效果與等待動畫。
- **動態色彩變換**：背景顏色與場景氛圍會隨著對話進度平滑過渡，增強沉浸感。
- **結果測驗圖匯出**：測驗結束可一鍵生成精美結果圖，方便使用者長按儲存與分享。
- **音效回饋**：內建背景音樂與點擊音效，提升互動體驗。
- **科學依據**：指標參考 GAD-7 焦慮量表。

## 🛠 技術棧

- **框架**：Vue 3 (Composition API, Router)
- **構建工具**：Vite + TypeScript
- **樣式與動畫**：Tailwind CSS v4 + 內建 CSS Animations
- **狀態管理**：Pinia
- **影像處理**：html-to-image (輸出高品質 2x 結果圖)
- **音效管理**：howler (全域靜音控制與背景音樂無縫循環)
- **部署方式**：GitHub Actions 自動部署至 GitHub Pages

## 🚀 快速開始

```bash
# 安裝依賴
pnpm install

# 啟動開發伺服器
pnpm dev

# 打包專案
pnpm build
```
